== Description ==
Alex Hashtag Grabber allows you to grab tweets containing a specific hashtag from Twitter and embed them in any post or page. 
 The best way to get fast support is by posting your question in a comment at http://anthony.strangebutfunny.net/my-plugins/twitter-hashtag-grabber/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `the directory "wordpress-sql-backup"` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Twitter Hashtags" under Settings in your dashboard for help and instructions
